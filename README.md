# Naija Rates

A savings interest rate comparison website for Nigerian fintech platforms. Enter an amount and duration to instantly compare gross interest, withholding tax (WHT), net interest, and total payout across multiple providers.

## Supported Platforms

| Platform  | Product       |
| --------- | ------------- |
| PiggyVest | SafeLock      |
| Cowrywise | Fixed Savings |
| OPay      | OPay Fixed    |
| Kuda      | Kuda Fixed    |
| PalmPay   | Fixed Savings |
| FairMoney | FairLock      |

## Disclaimer

Interest rates and product features are manually sourced from the respective fintech mobile applications. Rates are subject to change by providers at any time. All outcomes are estimates for comparison purposes only.
